# What can we do

## Design interface in minites

Dozens of beautiful components allow you to easily build perfect interactive pages by simply dragging and dropping the components.

![image](https://cdn.illacloud.com/official-website/img/solutions/dashboard/design.svg)

## Connect to any databases or APIs

Regardless of the database or API you use to manage data, you can quickly connect to your data source by entering the configuration information in ILLA. ILLA will access your data in a secure manner, ensuring the security of your data.

![image](https://cdn.illacloud.com/official-website/img/solutions/cms/data-source.svg)

The data sources we support include but are not limited to PostgreSQL, MySQL, MongoDB, Snowflake, Amazon S3, Google Sheets, Airtable, Stripe, etc.

## Integrate with AI Agents to analyze data

You can create an AI agent with data analysis or question-answering capabilities and seamlessly integrate it into your dashboard. This AI agent can assist you in analyzing data by providing insights and answering questions.

![image](https://cdn.illacloud.com/official-website/img/solutions/dashboard/analysis.gif)
